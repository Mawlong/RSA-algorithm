# RSA-Algorithm
An  implementation of RSA encryption algorithm in Python using GMPY2 and Pycrypto (to generate large prime) packages.

## Step-1: Install GMPY2:

Install GMPY2 using the following commands:

`brew install gmp`

`brew install mpfr`

`brew install libmpc`

`pip3 install gmpy2`

`pip3 install pycrypto`
